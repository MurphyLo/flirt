# flirt

***Flirting with MOOC, one at a time.***

## 功能概述

- [ ] 快捷键：对常用播放功能及其它所有功能提供支持。
- [ ] 视频倍速
- [ ] 字幕生成：基于讯飞 API 或 Whisper。
  - 检索定位：关键词匹配
  - 检索定位：语义检索
    - 基于 课件生成 结合视频图像进行增强
- [ ] 课件生成：对同一屏幕下的手写笔记进行增量。
  - 相邻屏幕定位
  - 同一屏幕的不同增量节点的手动切换 (1toN)
  - 同一屏幕内笔记：Markdown 格式支持
  - 视频笔记：可选区；节点内
  - (字幕生成3)
  - (播放列表2)
- [ ] 播放列表：基于文件夹构建。
  - 列表相邻视频跳跃，快速回溯(内存预配)
  - 课件节点间联系：注释、跳跃
